## 进程与线程  
进程就像车间，每个页面都是一个进程，而线程就是里面的生产线  
进程里面可以有一个或多个线程，线程不会单独存在  
多进程的好处就是同时打开多个页面，即使一个页面崩溃了也不影响其他页面的功能
多线程的好处就是可以并行处理任务
